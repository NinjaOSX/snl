# 超简单 Snell 一键安装脚本

本脚本可以自动安装 [Snell](https://manual.nssurge.com/others/snell.html)，以及在服务器上搭建高性能的科学上网环境。

## 📋 适用环境

- 支持的服务器：Linode、Vultr、搬瓦工等
- 支持的操作系统：CentOS 7、Ubuntu 18 以上版本

## 🚀 快速开始

#### 一键运行安装脚本

```bash
bash <(curl -s -L https://tinyurl.com/2gj5ucnd)
```
